Accessibility

Brief intro

I want everyone who uses my website to be able to find, read and understand my contents. Every reference is from W3C (2008) Web Content Accessibility Guidelines (WCAG) 2.0 Available at: https://www.w3.org/TR/WCAG20/ unless stated otherwise

Accessibility statement

Choice of colour:
For light background, dark coloured texts are used and for dark background, light coloured texts are used which increase the contrast between words and background so that users can view it clearly. (Rule 1.4.1).
W3C (2018) Rule 1.4.1 Use of Color
Available at: https://www.w3.org/TR/WCAG20/#visual-audio-contrast
(Accessed: 13/11/22)
Choice of text:
For mobile devices which are less than 600px, font-size is fixed at 18px. For desktop devices, font-size will increase as screen gets bigger which enables users to be able to view the contents clearly at any resolution. (Rule 1.4.4).
W3C (2018) Rule 1.4.4 Resize Text
Available at: https://www.w3.org/TR/WCAG20/#visual-audio-contrast
(Accessed: 13/11/22)
Page titles:
Every page has a large title indicating what's the page will present and it has headings and labels to give some information about the content will be about (Rule 2.4.2) and (Rule 2.4.6)
W3C (2018) Rule 2.4.2 Page Titled
Available at: https://www.w3.org/TR/WCAG20/#navigation-mechanisms
(Accessed: 13/11/22)
W3C (2018) Rule 2.4.6 Headings and Labels
Available at: https://www.w3.org/TR/WCAG20/#visual-audio-contrast
(Accessed: 13/11/22)
Link purposes:
Every link in the website has its own name which is related to its purpose and it has different appearance than other texts which allows users to easily indicate that it is a link, whether they have already accessed it and the name gives them some heads-up of what the webpage will be. (Rule 2.4.9)
W3C (2018) Rule 2.4.9 Link Purpose (Link Only)
Available at: https://www.w3.org/TR/WCAG20/#navigation-mechanisms
(Accessed: 13/11/22)
Abbreviations:
Full name is indicated before using abbreviation such as Sheffield Graduate Attributes (SGA). (Rule 3.1.4)
W3C (2018) Rule 3.1.4 Abbreviations
Available at: https://www.w3.org/TR/WCAG20/#meaning
(Accessed: 13/11/22)
Consistent Navigation:
Every page has the same style of headers and footers to allow users to navigate easily between pages. (Rule 3.2.3)
W3C (2018) Rule 3.2.3 Consistent Navigation
Available at: https://www.w3.org/TR/WCAG20/#consistent-behavior
(Accessed: 13/11/22)

Non-accessible content

Some pictures on Testing and Design Page may not be that accessible at smaller resolution. I'll be trying to get it fixed and for now these pictures can be handled using inbuilt browser zoom function which is CTRL + (Windows) or ZoomText

Enchance your experience

For laptops:
Magnifying text by using browser plugins- for example A+ Font SizeChangerLite for Chrome A+ FontSize Changer
Or use inbuilt browser zoom function which is CTRL + (Windows)
If you want to listen to certain sections or sentences, use Selection Reader
For phones:
Get magnification by changing browser default font size and display size in accessibility setting.
If you prefer higher contrast, use High Contrast extension on Google.

If you have any suggestion or feedback please fill in the form at Contact!
