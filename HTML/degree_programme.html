Degree programme

Let's see what I'm learning this year

COOM1001 Introduction to Software Engineering

Introduction of Software Engineering concepts that are needed to develop software systems that can meet basic functional requirements within a given problem domain.
It introduces the main steps in the process of developing such systems, from requirements analysis through to their implementation and testing. Teamworking is crucial in this module.

COOM1002 Foundations of Computer Science

Introduction of mathematical concepts and techniques which build the foundation of students in computing.
The main part of this module is to make students understand why these math topics are important through lectures and tutorials in order to prepare them for future computing courses.

COOM1003 Java Programming

The module introduces students programming concepts through Java programming language which is an object-oriented programming language.
Students will also understand the aspects of programming languages such as loop, array etc.
The important part is about how to write a well-structured and readable program to solve a particular problem.
This programming language is useful in building large software systems.

COOM1005 Machines and Intelligence

This module is about introduction to Artificial Intelligence. It also conveys the problems about AI field, for instance, whether computer is capable of understanding
It also provides a brief history of AI and problems faced in the past AI events.
This module also provides a lot of practical practice, for example, working with the Miro Robot which enables students to understand algorithms and data structures for an AI problem.

COOM1006 Devices and Networks

Introduction of how sophisticated architectures are constructed from basic elements of a computer.
It also introduces how computing can be linked into networks.
This module allows students to know more about electrical engineering and hardware such as logic gates and so on.

COOM1008 Web and Internet Technology

Web sites and Internet is a necessity to thehuman community these days. This module is important for students to know how to utilise the Internet.
It provides a lot of practical use of HTML, CSS, JavaScript to create responsive and yet aesthetic websites and conveys the importance of accessibility, legal and security issues to students so that they are aware of these issues when creating a website.

COOM1009 Introduction to Algorithms and Data Structures

This module has not yet commenced in this semester.
It is an introduction to the design and analysis of algorithms and data structures which is crucial to Computer Science.
Students learn how to quantify the efficiency of an algorithm and what algorithmic solutions are efficient.
Techniques for designing efficient algorithms are taught, including efficient data structures for storing and retrieving data.

FCE1001 Global Engineering Challenge Week

The project allows students to apply what they have learnt so far in providing real-life solutions which will help students to develop their employability and academic skills.
This project also requires students to work in group of 5 to 6 people which will provide them with experience of working with other people to develop a new, innovative solution to a problem just like how companies and organisations work.
