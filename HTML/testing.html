Testing

Optimisation

I have implemented and tried several ways to speed up my website speed.
Firstly, I have limited the number of images to be used for each page, for instance, a webpage of my website has no more than 4 images, except degree programme, design and testing pages.
Secondly, I have also reduced redirects where each page only has header navigation links, footer links and few external links except for SGA & mySkills, Accessibility and Legal & Security pages.
Lastly, I have kept the number of files to the minimum where only one CSS file is used and tried to keep my CSS file small.
As a result, my website has loaded faster than it used to be.
